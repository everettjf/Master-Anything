HTTP API Reference

This service exposes a small REST API. Read the authentication section before calling any endpoint.

Authentication

All requests must send a bearer token in the Authorization header. Tokens are issued by the login endpoint and expire after one hour.

Endpoints

Every endpoint requires authentication and returns JSON. Errors use standard HTTP status codes.

Rate Limits

Clients are limited to 100 requests per minute. Exceeding the limit on the endpoints returns a 429 response.
